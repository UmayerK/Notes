HTML Tag (<html>)

The <html> tag is the root element of an HTML page. This entire document is enclosed within <html> tags.

Head Tag (<head>)

The <head> tag contains meta information about the HTML page. It's not visible in the browser window.

Body Tag (<body>)

The <body> tag defines the document's body. It contains all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

Meta Tag (<meta>)

The <meta> tag provides metadata about the HTML document. Metadata is not displayed on the page but is machine parsable.

Div Tag (<div>)

This is a <div> element. It's often used as a container for other HTML elements.
